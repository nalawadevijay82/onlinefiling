File sales tax returns automatically with Avalara TrustFile for WooCommerce.

== Description ==

Avalara TrustFile for WooCommerce files your sales tax returns automatically with Avalara TrustFile service. This simple WooCommerce extension eliminates the need for you to upload your store's orders manually to Avalara TrustFile. The plug-in's auto sync feature allows you to sync orders automatically to your Avalara TrustFile account on daily basis. Simply automating your file return. The plug-in provides a simple user interface to easily configure the plug-in settings for your WooCommerce store.

Here is what you will do to file sales tax returns automatically with Avalara TrustFile service.

1. Registration
	* Register with [TrustFile](https://www.trustfile.com/ "TrustFile") 
	* Get the API token from [TrustFile](https://www.trustfile.com/ "TrustFile") advance settings

2. Order Sync
	* Install and activate Avalara TrustFile for WooCommerce plug-in into your WooCommerce shopping site
    * Configure integration settings and save it. That's all! 
    * With auto sync the plug-in will daily sync orders from your WooCommerce store to your Avalara TrustFile account.


== Installation ==

This section describes how to install Avalara TrustFile for WooCommerce plug-in and get it working.

1. Install WooCommerce plug-in.
2. Install Avalara TrustFile for WooCommerce plug-in either via the WordPress.org plug-in directory (search for “Avalara TrustFile”), or simply upload plug-in files to the `/wp-content/plug-ins/` directory on your server.
3. Activate the plug-in through the 'plug-ins' menu in WordPress.
4. Navigate to WooCommerce -> Settings -> Avalara TrustFile in your Admin Side Menu or WooCommerce -> Settings -> Integrations tab.
5. Enter your API Token into the "API Token" field and click 'Verify' button. The system will validate the API Token. "API Token Status" field will report the current status of the your API Token. If you do not have a API Token, login to your [TrustFile](https://www.trustfile.com/ "TrustFile") account to get your API token.
6. Check the box next to the "Enable order sync to TrustFile". 
7. Fill out the rest of your settings.
8. Click “Save Changes”.
